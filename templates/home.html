SecurePass API

Welcome to the SecurePass API, a tool designed to evaluate the security of your passwords with enhanced safety measures.

Using the API

To check if a password is secure, send a POST request to /check_password with a JSON payload. Include your password, name, date of birth (DOB), and email. The API evaluates your password against security criteria and personal info to ensure its strength.

POST /check_password

Expected JSON payload format:

{
  "password": "yourPassword123!",
  "name": "Your Name",
  "dob": "YYYY-MM-DD",
  "email": "your.email@example.com"
}

The API's response will indicate the password's security status and provide a detailed message.
